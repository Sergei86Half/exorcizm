# 🕯️ Ritual of Exorcism (Go Console Game)

Мрачный текстовый квест на языке Go, в котором вам предстоит провести обряд экзорцизма. Ошибка в ритуале может стать последней.

## 💀 Особенности
- **Атмосферный звук:** Использование библиотеки `Ebitengine` для воспроизведения эффектов (WAV).
- **Динамическая консоль:** Графика отрисовывается прямо в терминале с помощью ANSI-кодов.
- **Ритуал:** Строгая последовательность действий — используйте предметы в правильном порядке, чтобы выжить.

## 🛠️ Требования
Для работы игры необходим установленный [Go](https://go.dev) (версия 1.18+) и аудио-драйверы в системе:
- **Windows:** Не требует доп. драйверов.
- **Linux:** `libasound2-dev` (ALSA).
- **macOS:** AudioToolbox framework (встроен).
